# Requirements
1. Windows 10/11/Server >=2019 
2. Python >= 3.10

# Step 1: 
Install VigemBUS >= 1.20

# Step 2: 
Install python venv on Requirements.txt

# Step 3: 
Run `python gamepad.py`
